## ðŸš¨ Important Notes

### **Posts vs Events Differences**

| Feature | Posts | Events |
|---------|-------|--------|
| **Lifespan** | 24-48h (ephemeral) | Persistent |
| **Discovery Radius** | 5-10km | 25-50km |
| **Interactions** | Reactions ("I'm down", "Join me") | RSVPs |
| **Conversion** | Can evolve into Events | Cannot convert back |
| **Styling** | Casual, compact, gradient borders | Formal, larger, structured |

### **Conversion Thresholds**
- **5+ reactions**: Suggest conversion to host
- **10+ reactions**: Auto-convert to Event

### **Galaxy Theme**
- All UI must follow galaxy theme (glass cards, gradients, aurora)
- Mobile: Edge-to-edge, square corners
- Desktop: Rounded corners, proper padding
